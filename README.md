# ✈️ Goibibo Clone (zoibibo)

This is a simple front-end clone of the Goibibo website built using **React**.
The project includes pages like Flights, Hotels, Trains, Bus, Offers, Login/Signup, About, Terms, etc.

It is created only for learning and practice.

---

## Features

* React-based single page application
* Clean and simple UI
* Offers and Promo Code section
* About and Terms & Conditions pages
* Reusable components

## How to Run the Project

### 1. Go into the React project

```bash
cd react_clone_zoibibo
```

### 2. Install all dependencies

```bash
npm install
```

### 3. Start the development server

```bash
npm start
```

Now open:

```
http://localhost:3000
```

Your Goibibo Clone will run locally.
